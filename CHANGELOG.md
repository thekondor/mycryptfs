# Changelog

## [1.3.0-rc1] - 2026-03-01

### Breaking Changes
- Due to macOS restrictions, folders must be re-added in the configuration

### Changed
- core: reworked communication between the main app and FileProvider for immediate feedback
- core: support unlock using passwords stored in files
- ui: more descriptive error messages when encrypted folder is not available for various reasons
- installer: external script for checking running instances is removed (= no warnings on install)
